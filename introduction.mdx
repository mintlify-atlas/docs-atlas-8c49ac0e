## Welcome to Actividad-1

Actividad-1 is a Java console application that performs fundamental array operations and statistical calculations. This program demonstrates core programming concepts including array manipulation, user input handling, and mathematical computations.

## What Does It Do?

The program takes two arrays of integers as input from the user and performs two main operations:

1. **Element-wise Difference**: Calculates the difference between corresponding elements of the two arrays
2. **Average Calculation**: Computes the arithmetic mean of all input values across both arrays

## Who Is This For?

This program is designed for:

- **Java Students**: Learning fundamental array operations and control structures
- **Programming Beginners**: Understanding user input, loops, and basic calculations
- **Educators**: Teaching array manipulation and statistical computation concepts
- **Developers**: Reviewing basic Java syntax and Scanner usage

<Note>
  The program uses fixed-size arrays of length 7, making it ideal for demonstrating array concepts without complexity.
</Note>

## Core Concepts Demonstrated

Actividad-1 showcases several important programming concepts:

- **Array Declaration and Initialization**: Creating multiple integer arrays
- **Scanner Usage**: Reading user input from the console
- **Loop Structures**: Using for loops for array iteration
- **Arithmetic Operations**: Subtraction and division operations
- **Type Casting**: Converting integers to doubles for precise average calculation
- **Formatted Output**: Using `printf` for decimal formatting

## Program Output

When you run Actividad-1, you'll see:

1. Prompts to enter 7 values for the first array
2. Prompts to enter 7 values for the second array
3. The calculated average of all input values
4. A complete listing of the difference array showing each position and its value

<Warning>
  The program expects integer input. Non-integer values may cause input errors.
</Warning>
